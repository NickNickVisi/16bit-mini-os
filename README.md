A simple 16-bit real mode mini operating system that reads strings and displays them on screen using BIOS interrupts.

This is just a basic example and starting point for a 32-bit protected mode OS.

Prerequisites:

    - NASM
    - QEMU
    - GNU Make
    - GCC

Building the OS:

    - make build

Running the OS in QEMU:

    - make run

Cleaning up build files:

    - make clean
